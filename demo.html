Bootstrap 5 Notify Demo

This is a simple plugin that turns standard Bootstrap alerts into "Growl-like" notifications.

Please keep in mind that the master branch may contain bugs or broken code. Please use the link above if you are not able to debug and correct issues in the master branch. Thank you.

For icon, you can you user Fontawesome, Bootstrap Icon, or somethings you want...

For Animation, Please import animate.css, I used verion: v4

            // simple call
            $.notify("your message..." );
            
            
            // with header
             $.notify(
               {
                 title: 'Header Title',
                 message:  'your message...',
               } );
               
               
            // with icon
             $.notify(
               {
                 icon: 'bi bi-exclamation-circle',
                 title: 'Header Title',
                 message:  'your message...',
               } );
               
             // with Progress bar on
             $.notify(
               {
                 icon: 'bi bi-exclamation-circle',
                 title: 'Header Title',
                 message:  'your message...',
                 
               },{
               showProgressbar: true
               } );
            
             
           //Update
           var notify = $.notify('Saving Do not close this page...', { allow_dismiss: false });
           notify.update({ type: 'success', 'Success Your page has been saved!' });
           
           // Turn of Progress bar on
           $.notify('I have a progress bar', { showProgressbar: true });
           
           // Update Progress bar
           var notify = $.notify('Saving Do not close this page...', { allow_dismiss: false });
           notify.update({ type: 'warning', 'Oops Something happened. Correcting Now', progress: 20 });
